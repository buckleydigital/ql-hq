API Documentation

Use these endpoints with your API key to integrate with external tools.

Base URL: https://api.quoteleadshq.com/v1
Leads
GET /v1/leads — List leads (paginated, filterable by stage/ai_status/search)
POST /v1/leads — Create a lead (name, phone, email, etc.)
GET /v1/leads/:id — Get a single lead
PATCH /v1/leads/:id — Update a lead
Quotes
GET /v1/quotes — List quotes (filterable by status)
GET /v1/quotes/:id — Get a single quote with line items
SMS
POST /v1/sms — Send SMS to a lead (lead_id, message)
Appointments & Calls
GET /v1/appointments — List appointments (filterable by type)
GET /v1/voice-calls — List voice calls
GET /v1/pipeline — Pipeline stage counts
Authentication
All requests require an Authorization: Bearer <api_key> header.
Pagination: ?page=1&per_page=25 (max 100 per page).
Responses: { "data": [...], "meta": { "page", "per_page", "total" } }
